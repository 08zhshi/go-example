Go by Example 中文：通道

通道 是链接多个 Go 协程的管道。你可以从一个 Go 协程将值发送到通道，然后从别的 Go 协程中接收。

 
package main

import "fmt"

func main() {

使用 make(chan val-type) 创建一个新的通道。通道类型由他们传递的值决定。

 
	messages := make(chan string)

使用 channel <- 语法发送 一个新的值到通道中。这里我们在一个新的 Go 协程中发送 "ping" 到上面创建的messages 通道中。

 
	go func() { messages <- "ping" }()

使用 <-channel 语法从通道中 接收 一个值。这里我们将接收我们在上面发送的 "ping" 消息并打印出来。

 
	msg := <-messages
	fmt.Println(msg)
}

我们运行程序时，通过通道消息 "ping" 成功的从一个 Go 协程传到另一个中。

 
$ go run channels.go 
ping

默认发送和接收操作时阻塞的，直到发送方和接收方都准备好了。这个特性允许我们，不使用任何其它的同步操作来，在程序结尾等待消息 "ping"
